# Проект: Mesto
---
### Проект, рассказывающий о местах, которые посетил пользователь

Пользователь может редактировать свой профиль, добавлять и удалять карточки, а также открывать их для просмотра

## Используемые технологии
- HTML
- CSS
- JavaScript
